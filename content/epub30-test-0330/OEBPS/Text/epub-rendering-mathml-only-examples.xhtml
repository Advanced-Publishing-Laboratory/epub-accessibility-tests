Rendering MathML Only Tests

NOTE: You can ignore these tests if you are a non-visual reader.

math-050 Rendering inline MathML

The test passes if you are able to visually see the math equation (which should look similar to the reference image of the math equation).

The test fails if you are unable to see the MathML-rendered equation or the MathML rendering looks distorted.

Note: The equation may look a little different (i.e. not pixel perfect), but the content of the MathML-rendered math equation should be equivalent to the content in the PNG reference image.
