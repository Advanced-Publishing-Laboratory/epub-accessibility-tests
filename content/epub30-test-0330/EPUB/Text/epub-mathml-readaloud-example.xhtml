MathML Read-Aloud Tests

math-150 MathML Read-Aloud

The test passes if the following is true:

When you use the Reading Systems "Read Aloud" feature the mathML's alt-text is spoken.

You should hear the following: Mathml-alt-text: followed by the math equation

The test fails if you do not hear any math spoken.

Test 150

Math Equation will appear between the following two lines.

y − y 1 = y 2 − y 1 x 2 − x 1 ( x − x 1 )

Additional Notes for Test 150

If the test passes, please provide notes about what you heard. For example, did you hear the prefix "Mathml-alt-text:”. If you didn't hear that prefix before the spoken math please provide the math you hear spoken in your notes.
