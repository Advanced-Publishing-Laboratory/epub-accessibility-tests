MathML Displayed with Fallback PNG image in mathML Annotation / ARIA Label / Namespacing Tests

math-080 MathML Displayed with Fallback PNG image in MathML Annotation

The test passes if the assistive technology reports presence of the MathML expression and provides functionality to read it, for systems which can't read MathML this passes if the fallback annotation image is displayed and its alt-text description is announced.

The test fails if you do not see or hear any math equation

Test 080

Math Equation will appear between the following two lines.

y − y 1 = y 2 − y 1 x 2 − x 1 ( x − x 1 )

Additional Notes for Test 080

If the test passes, please provide notes about your MathML reading experience, and how well you were able to navigate in MathML. If you only had access to the annotation fallback PNG image did you see the png image and heard its description of "mathml annotation fallback alt text" followed by the equation.

Please make a note if you are able to see both the mathML rendered equation as well as its fallback png image.

math-090 MathML with Fallback of aria-label

The test passes if the assistive technology reports presence of the MathML expression and provides functionality to read it, for systems which can't read MathML this passes if the fallback aria-label description is announced.

The test fails if you do not see or hear any math equation

Test 090

Math Equation will appear between the following two lines.

y − y 1 = y 2 − y 1 x 2 − x 1 ( x − x 1 )

Additional Notes for Test 090

If the test passes, please provide notes about your MathML reading experience, and how well you were able to navigate in MathML. For example, did you hear the prefix "aria-label:” and you heard the expression “y minus y 1 equals StartFraction y 2 minus y 1 Over x 2 minus x 1 EndFraction left-parenthesis x minus x 1 right-parenthesis” or you heard the equation with no prefix.”

math-100 MathML using XML m: Namespacing

The test passes if the assistive technology reports presence of the MathML expression and provides functionality to read it.

The test fails if you are unable to navigate the MathML, or the MathML is not rendered visually correctly.

Test 100

Math Equation will appear between the following two lines.

y − y 1 = y 2 − y 1 x 2 − x 1 ( x − x 1 )

Additional Notes for Test 100

If the test passes, please provide notes about your MathML reading experience, and how well you were able to navigate in MathML. For example, did you hear the prefix "(mathml alt-text)” and you heard the expression “y minus y 1 equals StartFraction y 2 minus y 1 Over x 2 minus x 1 EndFraction left-parenthesis x minus x 1 right-parenthesis” or you heard the equation with no prefix.” Also note if the visual rendering was correct. (ie. did you see the entire equation or were parts missing.)
