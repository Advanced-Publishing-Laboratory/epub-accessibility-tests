Annotation Tests

Add a bookmark

[FUNDAMENTAL] anno-010

Add a bookmark

If you can add a bookmark via keyboard, the test passes.

[FUNDAMENTAL] anno-020

[FUNDAMENTAL] anno-030

Add a bookmark
