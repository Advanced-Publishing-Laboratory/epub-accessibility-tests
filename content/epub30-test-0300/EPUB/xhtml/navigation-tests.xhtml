Hyperlink navigation from within a page to location within the book

For the following tests:

  1. Navigate to and select the internal hyperlink.
  2. Verify that focus has moved to new location.
  3. Navigate to and select the Back hyperlink.
  4. Verify that focus is returned to original location.

[FUNDAMENTAL] nav-610

Hyperlink navigation from within a page to a location within the book

If you can navigation from this paragraph to the link target by selecting this link using the keyboard, the test passes.

You will be returned to this location by following the link back.
